<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced time tracking reports with export to CSV and PDF formats. Generate weekly summaries directly from the dashboard.

## How to Update

Follow these steps to ensure you run the latest version of BusinessAnalysis.io.

<Steps>
  <Step title="Check Current Version" icon="info">
    Open the settings menu in the top-right corner. View your version under `Account > About`.
  </Step>
  <Step title="Refresh App" icon="refresh-cw">
    Press `Ctrl+R` (or `Cmd+R` on macOS) to reload the latest build.
  </Step>
  <Step title="Clear Cache if Needed" icon="trash-2">
    In browser dev tools, clear site data for businessanalysis.io.
  </Step>
</Steps>

<Callout kind="tip">
  Enable auto-updates in your browser settings for seamless deployments. You receive notifications for major releases.
</Callout>

## Integration Examples

Use the new time tracking API in your custom scripts.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.businessanalysis.io/v1/time/entries', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_TOKEN}` },
    body: JSON.stringify({
      taskId: 'task-123',
      hours: 2.5,
      date: '2024-10-15'
    })
  });
  ```
  ```python
  import requests

  response = requests.post(
      'https://api.businessanalysis.io/v1/time/entries',
      headers={'Authorization': f'Bearer {YOUR_API_TOKEN}'},
      json={
          'taskId': 'task-123',
          'hours': 2.5,
          'date': '2024-10-15'
      }
  )
  ```
</CodeGroup>
