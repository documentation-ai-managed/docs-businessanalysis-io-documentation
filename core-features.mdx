## Task Creation and Tracking

You start by creating tasks directly from the dashboard. Assign them to team members, set due dates, and define priorities.

<Steps>
  <Step title="Create Task" icon="plus">
    Click the `+ New Task` button. Enter title, description, and assignee.

    ```javascript
    // Example API integration for task creation
    const newTask = await fetch('https://api.businessanalysis.io/tasks', {
      method: 'POST',
      headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
      body: JSON.stringify({
        title: 'Review quarterly report',
        dueDate: '2024-12-15',
        priority: 'high'
      })
    });
    ```
  </Step>
  <Step title="Track Progress" icon="eye">
    Use the progress bar to update status. Set reminders for overdue items.
  </Step>
  <Step title="Review Tasks" icon="list">
    Filter by status, assignee, or due date in the task list view.
  </Step>
</Steps>

<Callout kind="tip">
  Enable notifications in settings to stay updated on task changes without constant checking.
</Callout>

## Time Logging and Reports

Log time against tasks or projects effortlessly. Generate reports for billing or performance analysis.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Log time entry via API
  await fetch('https://api.businessanalysis.io/time-entries', {
    method: 'POST',
    body: JSON.stringify({
      taskId: 'task-123',
      hours: 2.5,
      description: 'Code review session'
    })
  });
  ```
  ```python
  # Log time entry
  import requests
  requests.post('https://api.businessanalysis.io/time-entries',
    json={
      'taskId': 'task-123',
      'hours': 2.5,
      'description': 'Code review session'
    },
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  ```
</CodeGroup>

<Expandable title="Generate Custom Reports" default-open="false">
  Select date range, tasks, and metrics. Export as PDF or CSV.

  Common report types include:
  - Billable hours summary
  - Productivity trends
  - Team utilization
</Expandable>

## Collaboration and File Sharing

Share files securely within projects. Real-time updates keep everyone aligned.

<Callout kind="info">
  All shared files support version history. You revert changes if needed.
</Callout>

Invite team members via email. Use `@mentions` in comments for direct notifications. Integrate with tools like Slack for seamless communication. These features ensure your team collaborates efficiently from anywhere.
